Summary:
This code analyzes different pools of the parameter spaces from the databases of results generated from the "HighCondParamSearch" simulations folder in various ways (i.e. number of models in particular states, average spike rates, parameter ranges for each pool, etc.).

Make sure to update all directory paths as they were possibly changed during the development of this Github repository.

1. To run the analyses and plot the results, go to the particular directory and open the "PlotResults.py" file in python

README:
Code by Alexandre Guet-McCreight
2015

Languague requirements:
NEURON (hoc)
Matlab

Summary:
This code looks at the spike time variability when you vary the magnitude of the intrinsic noise and the weight a single synapse at the soma.

Run Procedure:
1. Compile mod files (see NEURON Website for details on this: https://www.neuron.yale.edu/neuron/)
2. Run "SynchronousFiringTests.hoc" in NEURON.
3. Run "SynchronousFiringAnalysis.m" in Matlab.
